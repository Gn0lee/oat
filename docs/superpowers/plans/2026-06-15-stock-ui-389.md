# Stock UI 389 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Improve the stock main and stock analysis screens for #389 by replacing card-heavy layouts with metric, chart, and row/list screen grammar, and moving full market trends to a dedicated page.

**Architecture:** Keep existing API hooks and data calculations. Recompose pages with `components/layout/screen` primitives, split market trends into `/assets/stock/market`, and keep `/assets/stock` focused on household stock status and navigation.

**Tech Stack:** Next.js App Router, React client components, TanStack Query hooks, Tailwind CSS, shadcn/ui, Vitest + React Testing Library.
